# General Architecture Documentation

This repository contains general software architecture documentation, patterns, and best practices that are cloud-agnostic.

## Structure

```
architecture-general/
├── fundamentals/              # Core architectural concepts
│   ├── design-patterns/       # Software design patterns
│   └── architectural-styles/  # Microservices, monolith, etc.
├── distributed-systems/       # Distributed systems concepts
│   ├── messaging-patterns/    # Messaging and queuing patterns
│   └── event-driven/          # Event-driven architecture
├── domain-driven-design/      # DDD concepts and practices
├── networking/                # Network architecture concepts
├── security/                  # Security principles and patterns
├── observability/             # Monitoring, logging, tracing
│   └── metrics/               # Metrics and measurements
└── reference/                 # Reference materials and taxonomy
```

## Topics Covered

- **Design Patterns**: Event sourcing, CQRS, and other patterns
- **Distributed Systems**: Event-driven architecture, messaging patterns
- **Domain-Driven Design**: Event storming, bounded contexts
- **Observability**: Monitoring techniques, tools, and best practices
- **Security**: Authentication methods and security principles
- **Networking**: Hub-spoke architecture and network patterns
